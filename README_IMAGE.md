# Setup Welcome Image

## Cara Menambahkan Welcome Banner:

1. **Buat atau Download Image Welcome**
   - Ukuran recommended: 1280x720px atau 16:9 ratio
   - Format: PNG atau JPG
   - Nama file: `welcome.png`

2. **Simpan di Folder Project**
   - Letakkan file `welcome.png` di folder yang sama dengan `main.js`
   - Path: `c:\Users\Pongo\OneDrive\ドキュメント\BOT\BOT TELE RUST\welcome.png`

3. **Contoh Design Welcome Image:**
   - Background gradient/warna brand toko
   - Logo toko di tengah
   - Text "Selamat Datang" atau "Welcome"
   - Icon produk (opsional)
   - Desain yang menarik dan profesional

## Tools untuk Membuat Banner:

- **Canva** (https://canva.com) - Free, mudah digunakan
- **Figma** (https://figma.com) - Professional design tool
- **Photoshop** - Advanced editing
- **GIMP** - Free alternative to Photoshop

## Template Size:
- Width: 1280px
- Height: 720px
- Format: PNG dengan background
- File size: < 5MB (recommended < 1MB)

## Jika Tidak Ada welcome.png:
Bot akan otomatis fallback ke text message biasa tanpa gambar.
